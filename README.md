# 🍳 ChefOS

**Save Money. Waste Less. Cook Smarter.**

ChefOS is a Flutter-based mobile application designed to help users organize their kitchen inventory, reduce food waste, and make smarter cooking decisions. Instead of forgetting what is in the refrigerator or pantry, ChefOS keeps inventory organized while surfacing the ingredients that need attention first.

> **Status:** Phase 1 Complete 🚧  
> ChefOS is actively under development with additional features.
---
# 🎯 Why I Built ChefOS

Many people buy groceries without knowing what they already have at home, leading to unnecessary spending and food waste.

ChefOS was created to help users:

- reduce food waste
- save money
- organize kitchen inventory
- simplify meal planning

The long-term vision is to build an AI-powered kitchen assistant that provides transparent recommendations rather than guessing.

# 🔒 Privacy & Security

ChefOS currently stores inventory locally during development.

Future releases will implement secure cloud synchronization using Supabase with proper authentication and Row Level Security (RLS).

No API keys or sensitive credentials are committed to this repository.
